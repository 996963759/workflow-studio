# 工作流编辑器

这是一个本地运行的类 Dify 工作流应用 MVP，使用 React、TypeScript、Vite 和 React Flow 构建。

## 当前功能

- 可拖拽、可连线的可视化工作流画布
- 节点库：用户输入、大模型、知识检索、工具调用、条件分支、最终回答
- 右侧节点配置面板
- 根据节点输出变量自动生成变量列表
- 模拟逐节点运行的运行预览
- 保存到浏览器本地，并在刷新后自动恢复
- 当前工作流 JSON 导入和导出
- 重置为示例工作流
- 响应式三栏产品界面

## 使用说明

- 点击左侧节点库可以新增节点。
- 在画布中拖拽节点并连接节点端点可以调整流程。
- 点击节点后，在右侧节点配置中修改名称、用途、提示词和输出变量。
- 点击保存会写入当前浏览器的 `localStorage`，下次打开会自动恢复。
- 点击导出会下载当前工作流 JSON 文件。
- 点击导入可以载入之前导出的 JSON 文件。
- 点击重置会恢复为内置示例工作流。

## 本地运行

```powershell
npm.cmd install
npm.cmd run dev
```

## 构建

```powershell
npm.cmd run build
```

## 当前边界

这是前端 MVP。当前还没有接真实大模型 API、数据库、用户系统和外部工具执行器。确认编辑器形态后，下一步应增加后端 API 层来处理保存、运行和模型调用。
