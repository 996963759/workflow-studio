# 客服知识库示例

## 退款规则

用户在购买后 7 天内可以申请退款。已经使用超过 50% 服务额度的订单，需要人工复核后再处理。退款通常会在 3 到 5 个工作日内原路返回。

## 发票规则

企业用户可以在订单完成后申请增值税普通发票。申请时需要提供公司名称、税号、邮箱和订单号。发票通常会在 2 个工作日内发送到邮箱。

## 账号安全

如果用户反馈账号异常登录，应先建议用户修改密码并开启二次验证。客服需要记录异常登录时间、登录地区和用户联系方式，再提交安全工单。

## API 调用失败

当用户反馈 API 调用失败时，需要先确认错误码、请求时间、接口路径和请求 ID。常见原因包括 Key 无效、额度不足、参数格式错误和网络超时。
